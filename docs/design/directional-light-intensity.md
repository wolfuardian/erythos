# feat/directional-light-intensity

## 需求
在 Properties 面板為 Directional Light 新增 intensity 數值輸入欄位，讓使用者可調整光源亮度。

## Scope
- `src/panels/properties/` — DirectionalLight 屬性區塊新增 intensity 欄位
- `src/core/commands/` — 新增或擴充 SetLightIntensityCommand（走 Command 模式保 undo/redo）

## 風險 / Trade-off
- intensity 變更必須走 `editor.execute(command)`，直接改 Three.js object 會繞過 undo/redo 歷史

## 不做的事
- 不動其他光源類型（PointLight、SpotLight、AmbientLight）
- 不做 intensity 動畫或曲線編輯
- 不改 UI layout 以外的 Properties 面板結構
