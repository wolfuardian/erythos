# feat/project-folder-icons

## 需求
在 Project Panel sidebar 的資料夾節點名稱左側加上資料夾 icon，風格仿 Unity Editor。

## Scope
- `src/panels/project/` — sidebar 樹狀元件與對應 CSS Module
- `src/assets/` 或 `src/panels/project/icons/` — 資料夾 icon SVG（若需新增）

## 風險 / Trade-off
- icon 來源方式（inline SVG / `<img>` / CSS background）需與現有 icon 用法對齊，否則後續擴充風格不一致
- Unity style 資料夾 icon 有展開/折疊兩種狀態，需確認是否兩種都做

## 不做的事
- 不為非資料夾的檔案節點加 icon（如 mesh、material）
- 不改資料夾展開/折疊動畫
- 不引入 icon library（保持輕量）
