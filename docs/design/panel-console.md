# feat/panel-console

## 需求
新增 Console panel module，能在編輯器 UI 中顯示 log 訊息。

## Scope
- `src/panels/console/` — 新模組（ConsolePanel 元件、樣式、log store）
- `src/app/App.tsx` — 註冊 Console panel 至 Dockview

## 風險 / Trade-off
- log 資料流設計（SolidJS signal store vs event bus）需確定，影響後續擴充性
- log 量大時若不做 virtualization，DOM 節點數可能爆炸；初版可先設上限（如最多 500 條）

## 不做的事
- 不實作 log filter / search
- 不實作 log level 顏色分級以外的進階功能
- 不接 external logging library
- 不做 log persistence（重整即清空）
