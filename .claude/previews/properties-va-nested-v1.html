子群組微底色（tinted body）。
  • Data 整塊鋪 --bg-section（比 panel 深一階）
  • 父 OBJECT 仍維持 --bg-panel，對比出「區塊感」
  • 階層最強、但視覺權重也最重；子群組越多畫面越雜
DNA 守則： 四 panel 保留 Variant A 所有既定規則 — foldout 三角透明標題（--text-secondary、uppercase）、 輸入 rest 僅 border-bottom、 focus 為 2px 藍底線 + 1px blue glow（color-mix 40%）、 X/Y/Z 為 16px 紅/綠/藍 badge + 白字、字級 --font-size-sm。 三種巢狀手法僅在「子群組如何從父分離」一題上做差異：Indent = 空間位移；Rail = 左側一條線；Tint = 背景深一階。 所有色值嚴守 token，唯一 #fff 為 badge 對比白字（同前版例外）。
