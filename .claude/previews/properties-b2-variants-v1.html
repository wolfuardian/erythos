工業深黑 + mono 數值。
  • Group title 粗體 uppercase + wider letter-spacing（1px）
  • 分組 body 使用最深 --bg-app（比 Baseline 更黑）
  • 所有數值、X/Y/Z 前綴用 --font-mono；focus 為 2px 加粗藍邊 + halo
Token 依據： 所有色值對應 src/styles/theme.css（無 ad-hoc hex）。 背景 --bg-app / --bg-panel / --bg-section / --bg-header / --bg-input / --bg-input-focus； 文字 --text-primary / --text-secondary / --text-muted； 軸色 --accent-red / --accent-green / --accent-blue； 邊 --border-subtle / --border-medium / --border-focus（= accent-blue）。 Focus halo / Variant B 填充 focus 使用 color-mix() 混合 accent-blue 與基色。 Variant A badge 白字為 #fff 前景（badge 本身為 token 色）；其他四版不出現任何 ad-hoc hex。
