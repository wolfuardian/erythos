Panel 背景色系統一 — 3 方案 mockup
DV 審計發現 Scene tab（暖灰）/ Project tab（冷藍 cast）/ 浮窗（近黑）三套色系並存。
本檔出 3 方案：A（保守、不動 token）/ B（中庸、加 --bg-overlay）/ C（激進、整體下調明度 + 加 --bg-overlay）。
方案 A · 全對齊 token
不動 theme.css。浮窗 / toolbar 借用既有 --bg-app（#1a1a1a），比 panel 暗一階。
明度階梯 #1a → #24 → #2a → #33（短而漸進）。
